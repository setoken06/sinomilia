# シノミリア

# ルール
概要

プレイヤーは手札のカードを1枚ずつ伏せ、チップを交互に積む。
積まれたチップの合計数に近い数字のカードを伏せたプレイヤーが、積んだチップを獲得する。
どちらかがチップを失うか、残りの手札が2枚になった時点でゲーム終了。より多くのチップを獲得したプレイヤーの勝利！


ゲームの準備

0から9までの数字カードを持つ
それぞれのプレイヤーは15枚ずつ持つ
数字カードは相手に見えないよう手札として持っておく。（＝手札）
スタートプレイヤーはランダムに決定する

ラウンドの流れ

お互いに手札から数字カードを1枚選び、裏向きして置く。
スタートプレイヤーから交互にアクションを行うか、パスを宣言する。（＝後述「手番の流れ」を参照）


ラウンドの終了条件

1. テーブルの中央に９枚目のチップが積まれたとき
2. 両プレイヤーがどちらも連続してパスをしたとき
3. チェンジを行った直後にパスを行った場合


ラウンドの終了時、最後にアクションを行なったプレイヤーから自分の伏せている数字カードを公開し、その1秒後にもう片方のプレイヤーも公開する。

積まれたチップの合計数に近い数字のカードを伏せたプレイヤーが、積んだチップを獲得する。

差が同じだった場合、最後のアクションを行なったプレイヤーが敗北となる。（パスはアクションに含まれない）
どちらのプレイヤーもチップを積まずに同点になった場合は、最初にパスをしたプレイヤーがチップを獲得する。

どちらかがチップを失うか、残りの手札が2枚になった時点でゲーム終了。より多くのチップを獲得したプレイヤーの勝利！

同点の場合は、最終ラウンドに勝利したプレイヤーの勝利！

勝利できるチップの枚数

勝利ボーナス2枚

積まれたチップのうち「月」で出したチップ1枚につき
所持チップが0枚だった場合、5枚
所持チップが1枚だった場合、2枚

手番の流れ

スタートプレイヤーから交互に下記のアクションのうち１つを行う。

1. 自分の所持するチップを1枚「太陽」か「月」の面を表にしてテーブルの中央に置く。
2. チェンジ: 自分の所持するチップを1枚相手に渡すことで、ラウンドの初めに伏せたカードを1ラウンド中に1度のみ変更することができる。
3. パスを宣言する
